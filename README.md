# CS-340 Project Two: Grazioso Salvare Rescue Dashboard

## Project Overview

This project focused on developing an interactive dashboard for Grazioso Salvare using Python, MongoDB, Dash, and Plotly. The dashboard allows users to search, filter, and visualize animal shelter data through tables and charts. Throughout this project, I connected the dashboard to the MongoDB database using the CRUD module that I developed in Project One. This project gave me valuable experience building a complete client/server application and applying the concepts learned throughout the course.

## Reflection

### How do you write programs that are maintainable, readable, and adaptable?

One of the biggest lessons I learned during this project is the importance of separating different parts of an application into reusable modules. Instead of placing all of the database operations inside the dashboard, I created a separate CRUD Python module in Project One and reused it in Project Two. This made my code much easier to read, organize, and maintain. If changes are needed in the future, I can update the CRUD module without modifying the entire dashboard. Using descriptive variable names, functions, and modular design also made the project easier to test and debug.

### How do you approach a problem as a computer scientist?

I learned that solving software problems is much easier when I break larger tasks into smaller pieces. Before writing any code, I carefully reviewed the project requirements and planned how each feature would work. I tested each section individually before combining everything into the final dashboard. This step-by-step approach helped me identify problems early and made debugging much more manageable. I plan to continue using this process in future software development projects.

### What do computer scientists do, and why does it matter?

Computer scientists create software that helps organizations solve real-world problems by improving efficiency and supporting better decision-making. In this project, the dashboard allows Grazioso Salvare to quickly search, filter, and visualize animal shelter data instead of manually reviewing thousands of records. Having accurate and organized information makes it easier for employees to make informed decisions and complete their work more effectively. This project helped me understand how software can make a meaningful impact in real-world organizations.
